支付
===========

## Use Case Name

支付

## Brief Description

这个用例描述寄件人如何进行支付

## Flow of Events

#### Basic Flow

这个用例开始于寄件人填写完快递单后

1. 用户选择支付方式：

	- 线上支付：网络银行，第三方支付平台等
	- 线下支付：银行卡，现金

2. 用户完成支付
