查询快递信息
===========

## Use Case Name

查询快递信息

## Brief Description

这个用例描述用户如何查询快递信息

## Flow of Events

#### Basic Flow

这个用例开始于用户需要查询快递信息时

1. 用户打开查询页面
2. 用户输入订单号
3. 网站返回查询结果

#### Alternative Flows

- 订单号不存在

	1. 提醒用户订单号不存在
	2. 重新跳转到查询页面
